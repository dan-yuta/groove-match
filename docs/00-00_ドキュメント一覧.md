## 読み方ガイド

### Markdownファイル（.md）
以下のいずれかの方法で閲覧できます。
- **VS Code** で開く → プレビュー機能（Ctrl+Shift+V）で見やすく表示
- **GitHub** にpushすると自動でフォーマットされて表示
- テキストエディタでそのまま開いても読めます

### HTMLファイル（.html）
- ファイルをダブルクリック、またはブラウザにドラッグ&ドロップで開けます
- インターネット接続不要（すべてインライン CSS で完結）
