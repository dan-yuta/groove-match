## ルーティング構造

/                    → ランディングページ (公開)
├── /login           → ログイン (auth グループ)

├── /dashboard       → ダッシュボード (main グループ, 要認証)

## モバイルアプリ構成

Capacitor を使用して、Next.js で構築した Web アプリを iOS / Android ネイティブアプリとしてラップしています。

### 基本アーキテクチャ

- **Capacitor** がネイティブの WebView 内で Web アプリを表示
- モバイルビルド時は Next.js の静的エクスポート（`output: 'export'`）を使用
- `src/lib/platform.ts` でプラットフォーム（Web / iOS / Android）を検出し、プラットフォーム固有の処理を分岐

### Safe Area 対応

ノッチ付きデバイス（iPhone X以降等）に対応するため、Safe Area Insets を CSS で適用しています。

### ネイティブプラグイン

| プラグイン | 用途 |
|-----------|------|
| StatusBar | ステータスバーのスタイル制御 |
| SplashScreen | スプラッシュスクリーン表示制御 |
| PushNotifications | プッシュ通知の受信・処理 |
| Haptics | 触覚フィードバック（タップ・操作時） |
| Keyboard | キーボード表示/非表示のハンドリング |
