# マッチングアルゴリズム詳細

## スコア算出ロジック

### 総合スコア = 地域 + 楽器 + ジャンル + スキル + スケジュール (最大155pt → 100に正規化)

---

### 1. 地域スコア (最大50pt)

| 条件 | スコア |
|------|--------|
| 同じ都道府県 + 同じ市区町村 | 50pt |
| 同じ都道府県のみ | 35pt |
| 異なる都道府県 | 0pt |

### 2. 楽器スコア (最大25pt)

補完的な楽器構成を高く評価（バンド結成に有利）

| 条件 | スコア |
|------|--------|
| 完全に異なる楽器 | 25pt |
| 一部重複 | 15pt |
| 全て同じ楽器 | 5pt |

### 3. ジャンルスコア (最大30pt)

```
スコア = (共通ジャンル数 / 自分のジャンル数) × 30
```

### 4. スキルスコア (最大15pt)

近いスキルレベル同士を高く評価

```
レベル差 = |自分のスキル値 - 相手のスキル値|
スコア = max(0, 15 - レベル差 × 5)

スキル値: beginner=1, intermediate=2, advanced=3, expert=4
```

| レベル差 | スコア |
|---------|--------|
| 0 (同じ) | 15pt |
| 1 | 10pt |
| 2 | 5pt |
| 3 | 0pt |

### 5. スケジュールスコア (最大35pt)

```
基本スコア = (共通曜日数 / 自分のスケジュール数) × 35
ボーナス = 共通曜日で時間帯が重複している場合 +5pt/曜日
```
