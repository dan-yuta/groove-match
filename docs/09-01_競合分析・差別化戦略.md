# BandMatch 競合分析・差別化戦略

> **本アプリのゴールは「コピバンを組んでライブに出ること」です。**
> メインフローは「仲間探し → コピバン結成 → ライブ出演」。
> 練習トラッカーやフレンド機能は、そのゴールに到達するまでの挫折を防ぐサポート機能として位置づけています。

## 1. 競合サービス概要

### OURSOUNDS（https://oursounds.net/）

| 項目 | 内容 |
|------|------|
| 形態 | バンドメンバー募集 掲示板（Web） |
| 運営歴 | 2005年〜（20年以上） |
| 料金 | 無料 |
| 主な機能 | 募集記事の投稿・検索、メンバー検索、<br>メッセージ、お気に入り |
| 検索方法 | 都道府県別、テーマ別（初心者・シニア等）で絞り込み |
| ターゲット | 全年代・全レベル |
| UI | 従来型の掲示板スタイル |

**強み:**
- 無料で利用可能
- 20年以上の運営実績でユーザー数が多い
- 不正ユーザーの徹底排除に注力
- 都道府県別・テーマ別の検索が可能

**弱み:**
- UIが古い掲示板スタイルで、若年層には馴染みにくい
- マッチング機能なし（自力で検索するのみ）
- 挫折防止やモチベーション維持の仕組みがない
- バンド結成後のサポートがない
- 用が済んだら使わなくなる
  （継続利用の動機がない）

---

### ROCKJOINT（https://rockjoint.link/）

| 項目 | 内容 |
|------|------|
| 形態 | オフラインイベント運営 + マッチングアプリ |
| 実績 | 累計4,601名参加 |
| 料金 | イベント参加費（詳細非公開） |
| 主な機能 | バンドメンバー募集交流会、<br>バンドサークル「コピフェス」、<br>ライブイベント |
| 展開地域 | 東京・大阪・京都・神戸・横浜・福岡 |
| ターゲット | 初心者〜経験者 |
| マッチングアプリ | 1,000名以上登録 |

**強み:**
- リアルイベントでの対面交流（実際に会える）
- ライブ出演機会を直接提供
- プロアーティストとのコラボ実績（DOESなど）
- 累計参加者数の実績
- メンバー募集から演奏機会まで一貫提供

**弱み:**
- 対面前提のため地方ユーザーには弱い
- オンライン機能が弱い（アプリは補助的）
- 継続的な関係構築の仕組みが薄い
- 料金情報の透明性が低い
- イベント日程に合わせる必要がある

## 4. BandMatch の差別化ポイント

### 4.1 競合にないもの（BandMatchだけの強み）

| 差別化要素 | 詳細 | 競合との違い |
|-----------|------|-------------|
| **コピバン特化** | コピバンに完全特化したサービス設計 | OURSOUNDS/ROCKJOINT=オリジナル・コピー混在で、<br>コピバン向けの機能がない |
| **曲ベースマッチング** | やりたい曲・好きなアーティストでスコアリング。<br>「天体観測やりたい人」を即発見 | 競合には曲ベースの検索/マッチング機能がない |
| **セットリスト管理** | コピバンのセットリストをアプリ内で管理・共有 | 競合にはバンド管理機能自体がない |
| 自動マッチング | 地域・楽器・ジャンル・スキル・スケジュール・曲の<br>6要素でスコアリング | OURSOUNDS=自力検索、<br>ROCKJOINT=対面のみ |
| 挫折防止の仕組み | Duolingo型ゲーミフィケーション<br>（ストリーク・バッジ・ランキング・通知）で<br>継続を強力に支援 | 競合2社にはこの概念がない |
| **練習トラッキング** | 曲ごとの練習時間記録、ストリーク、<br>週間目標、バッジ12種 | 競合に練習支援機能なし |
| **フレンド＆ランキング** | フレンド間の練習状況共有、<br>週間ランキング、応援機能 | OURSOUNDSにはフレンド機能なし、<br>ROCKJOINTはイベント限定 |
| **SNS連携** | X / Instagram / LINE / YouTube<br>アカウント連携 | 競合にSNS連携なし |
| 一気通貫サポート | 曲マッチング→コピバン結成→<br>セットリスト管理→イベント登録が<br>1アプリで完結 | OURSOUNDS=募集まで、<br>ROCKJOINT=イベントのみ |
| モダンUI/UX | ダークモード・グラスモーフィズム・<br>レスポンシブ | OURSOUNDSは掲示板UI |
| スケジュールマッチング | 練習可能な曜日・時間帯で絞り込み | 競合は地域とジャンルだけ |
| 楽器スロット制 | コピバンに「何が足りないか」が一目瞭然 | 掲示板では文章で<br>「ギター募集」と書くだけ |
| 初心者特化 | 初心者歓迎バッジ、<br>スキルレベル別マッチング | 競合は全レベル混在で<br>初心者が萎縮しがち |
| クロスプラットフォーム | Web・iOS・Androidで同じ体験を提供 | OURSOUNDSはWeb掲示板のみ |
| プラットフォーム対応料金戦略 | Web(Stripe 3.6%)・iOS/Android(ストア手数料15%)を併用し、<br>手取りを最適化 | 競合は料金戦略の言及なし |

### 4.2 差別化の3つの柱

> **本アプリのゴールは「コピバンを組んでライブに出ること」。** 3つの柱はこのゴールを実現するために設計されています。

#### 柱1: 曲ベースのスマートマッチング（仲間探し）
- **「やりたい曲が同じ仲間が、すぐ見つかる」** がコアバリュー
- 6要素スコアリング（曲マッチングが最重要ウェイト）で「探す手間」をゼロに
- 曲名・アーティスト名で直接検索 → 同じ曲をやりたい人がすぐ見つかる
- Freeプランでも1日3回のマッチングが可能

#### 柱2: コピバン結成→ライブまで一気通貫
- 曲マッチングで同じ曲をやりたい仲間を見つける
- コピバン作成＋セットリスト管理でチームを組む
- イベントボードでコピバンライブを見つけてエントリー
- 1つのアプリで完結（複数サービスを行き来しなくてよい）
- **「仲間探し → コピバン結成 → ライブ出演」がメインフロー**

#### 柱3: Duolingo型挫折防止エコシステム（サポート機能）

メインゴール「コピバンを組んでライブに出る」に到達するまでの挫折を防ぐサポート機能群です。
Duolingoの語学学習と同様に、毎日の小さな練習を継続する仕組みで挫折を防止します。

- **練習トラッカー**:
  曲ごとの練習時間を記録。
  ストリーク（連続練習日数）で継続を可視化
- **バッジ＆マイルストーン**:
  12種類のバッジ（7日連続練習、累計100時間、初ライブ等）で成長を実感
- **フレンドランキング**:
  フレンド間の週間練習ランキングで、仲間と切磋琢磨
- **通知システム**:
  練習リマインド、ストリーク危機通知、フレンド活動通知でモチベーション維持
- **SNS連携**:
  X / Instagram / LINE / YouTube アカウント連携で、バンド活動のアピールも簡単
- コミュニティのタイムラインで孤独感を解消
- 同じレベルの仲間と繋がれるスキルマッチング

## 5. 今後強化すべき機能（競合との差をさらに広げる）

| 機能 | 優先度 | 差別化効果 | 詳細 |
|------|--------|-----------|------|
| プッシュ通知 | 高 | 競合にない | マッチング通知・メッセージ通知をリアルタイム配信 |
| リアルタイムチャット | 高 | OURSOUNDSのメッセージ機能を超える | マッチング後の即座のコミュニケーション |
| 練習マッチング | 高 | 競合にない | 「今週末セッションしたい人」のリアルタイム募集 |
| 成長トラッキング | 中 | 競合にない | 練習時間の推移グラフ、スキルレベルの自動更新提案 |
| バンド活動カレンダー | 中 | 競合にない | 練習日程・イベント日程の共有カレンダー |
| レコメンドエンジン | 中 | OURSOUNDSの検索を超える | 行動履歴ベースのおすすめ（Amazon Personalize） |
| 動画プロフィール | 低 | 演奏力が事前に分かる | 自己紹介動画・演奏動画のアップロード |
| オンラインセッション | 低 | ROCKJOINTの地域制約を超える | WebRTCでリモートセッション |

## 6. キャッチコピー案

### メインコピー
> **「やりたい曲で仲間を見つけて、コピバン組んで、ライブに出よう」**

### サブコピー候補
- 「『天体観測』やりたい人、3人見つかりました ── BandMatchで、コピバンライブデビューしよう」
- 「好きな曲でつながる → コピバン結成 → ライブ出演。全部このアプリで」
- 「掲示板で探す時代は終わり。やりたい曲を登録するだけで仲間が見つかる」
- 「仲間探しからライブまで一気通貫。BandMatchがぜんぶサポート」

### 競合との違いを一言で
- **vs OURSOUNDS:** 「曲名で検索できる。同じ曲をやりたい人だけが見つかる」
- **vs ROCKJOINT:** 「イベントを待たなくていい。やりたい曲を登録するだけで仲間が見つかる」
