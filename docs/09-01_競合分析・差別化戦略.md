# BandMatch 競合分析・差別化戦略

## 1. 競合サービス概要

### OURSOUNDS（https://oursounds.net/）

| 項目 | 内容 |
|------|------|
| 形態 | バンドメンバー募集 掲示板（Web） |
| 運営歴 | 2005年〜（20年以上） |
| 料金 | 無料 |
| 主な機能 | 募集記事の投稿・検索、メンバー検索、メッセージ、お気に入り |
| 検索方法 | 都道府県別、テーマ別（初心者・シニア等）で絞り込み |
| ターゲット | 全年代・全レベル |
| UI | 従来型の掲示板スタイル |

**強み:**
- 無料で利用可能
- 20年以上の運営実績でユーザー数が多い
- 不正ユーザーの徹底排除に注力
- 都道府県別・テーマ別の検索が可能

**弱み:**
- UIが古い掲示板スタイルで、若年層には馴染みにくい
- マッチング機能なし（自力で検索するのみ）
- 挫折防止やモチベーション維持の仕組みがない
- バンド結成後のサポートがない
- 用が済んだら使わなくなる（継続利用の動機がない）

---

### ROCKJOINT（https://rockjoint.link/）

| 項目 | 内容 |
|------|------|
| 形態 | オフラインイベント運営 + マッチングアプリ |
| 実績 | 累計4,601名参加 |
| 料金 | イベント参加費（詳細非公開） |
| 主な機能 | バンドメンバー募集交流会、バンドサークル「コピフェス」、ライブイベント |
| 展開地域 | 東京・大阪・京都・神戸・横浜・福岡 |
| ターゲット | 初心者〜経験者 |
| マッチングアプリ | 1,000名以上登録 |

**強み:**
- リアルイベントでの対面交流（実際に会える）
- ライブ出演機会を直接提供
- プロアーティストとのコラボ実績（DOESなど）
- 累計参加者数の実績
- メンバー募集から演奏機会まで一貫提供

**弱み:**
- 対面前提のため地方ユーザーには弱い
- オンライン機能が弱い（アプリは補助的）
- 継続的な関係構築の仕組みが薄い
- 料金情報の透明性が低い
- イベント日程に合わせる必要がある

## 4. BandMatch の差別化ポイント

### 4.1 競合にないもの（BandMatchだけの強み）

| 差別化要素 | 詳細 | 競合との違い |
|-----------|------|-------------|
| 自動マッチング | 地域・楽器・ジャンル・スキル・スケジュールの5要素でスコアリング | OURSOUNDS=自力検索、ROCKJOINT=対面のみ |
| 挫折防止の仕組み | 練習ログ・マイルストーン・コミュニティで継続モチベーション維持 | 競合2社にはこの概念がない |
| 一気通貫サポート | マッチング→バンド管理→イベント登録が1アプリで完結 | OURSOUNDS=募集まで、ROCKJOINT=イベントのみ |
| モダンUI/UX | ダークモード・グラスモーフィズム・レスポンシブ | OURSOUNDSは掲示板UI |
| スケジュールマッチング | 練習可能な曜日・時間帯で絞り込み | 競合は地域とジャンルだけ |
| 楽器スロット制 | バンドに「何が足りないか」が一目瞭然 | 掲示板では文章で「ギター募集」と書くだけ |
| 初心者特化 | 初心者歓迎バッジ、スキルレベル別マッチング | 競合は全レベル混在で初心者が萎縮しがち |

### 4.2 差別化の3つの柱

#### 柱1: スマートマッチング
- 5要素スコアリングで「探す手間」をゼロに
- Freeプランでも1日3回のマッチングが可能
- フィルター（地域・楽器・ジャンル・スキル）で精度向上

#### 柱2: 挫折防止エコシステム
- 練習ログで日々の努力を可視化
- マイルストーンで成長を実感（初セッション→初ライブ→バンド結成）
- コミュニティのタイムラインで孤独感を解消
- 同じレベルの仲間と繋がれるスキルマッチング

#### 柱3: バンド結成→ライブまで一気通貫
- マッチングで仲間を見つける
- バンド管理機能でチームを組む
- イベントボードでライブを見つけてエントリー
- 1つのアプリで完結（複数サービスを行き来しなくてよい）

## 5. 今後強化すべき機能（競合との差をさらに広げる）

| 機能 | 優先度 | 差別化効果 | 詳細 |
|------|--------|-----------|------|
| リアルタイムチャット | 高 | OURSOUNDSのメッセージ機能を超える | マッチング後の即座のコミュニケーション |
| 練習マッチング | 高 | 競合にない | 「今週末セッションしたい人」のリアルタイム募集 |
| 成長トラッキング | 中 | 競合にない | 練習時間の推移グラフ、スキルレベルの自動更新提案 |
| バンド活動カレンダー | 中 | 競合にない | 練習日程・イベント日程の共有カレンダー |
| レコメンドエンジン | 中 | OURSOUNDSの検索を超える | 行動履歴ベースのおすすめ（Amazon Personalize） |
| 動画プロフィール | 低 | 演奏力が事前に分かる | 自己紹介動画・演奏動画のアップロード |
| オンラインセッション | 低 | ROCKJOINTの地域制約を超える | WebRTCでリモートセッション |

## 6. キャッチコピー案

### メインコピー
> **「ギター始めたばかりでも、3ヶ月でライブに出られる」**

### サブコピー候補
- 「仲間探しから初ライブまで、BandMatchがぜんぶサポート」
- 「スキルが近い仲間と出会える、初心者のためのバンドマッチング」
- 「掲示板で探す時代は終わり。AIがあなたにぴったりのメンバーを見つけます」
- 「練習ログを記録して、仲間と一緒に成長しよう」

### 競合との違いを一言で
- **vs OURSOUNDS:** 「探すのではなく、出会える」
- **vs ROCKJOINT:** 「イベントを待たなくていい。今すぐ仲間が見つかる」
