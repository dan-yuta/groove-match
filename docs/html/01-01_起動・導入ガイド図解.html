🎸 このアプリは何？

BandMatch は「好きな曲をコピーしたい初心者が同じ曲をやりたい仲間を見つけてコピバンを組み、ライブに出る」ことをゴールとするコピバン特化の Web / iOS / Android アプリです。
メインフローは仲間探し → コピバン結成 → ライブ出演。
Duolingo型の練習トラッカーやフレンド機能は、そのゴールに到達するまでの挫折を防ぐサポート機能です。

👩‍🎸
課題
仲間がいない、レベルが合わない、ライブに出られない → 半年で挫折
💡
解決策
仲間探し→コピバン結成→ライブ出演を一気通貫サポート＋Duolingo型挫折防止

🔍 前提ツールの確認

以下のツールがPCにインストールされているか確認してください

ツール必要バージョン確認コマンド入手先

1

📦 ソースコードの取得

GitHubからリポジトリをクローンします

Terminal # リポジトリをクローン git clone https://github.com/dan-yuta/band-match.git # プロジェクトディレクトリに移動 cd band-match

📦 依存パッケージのインストール

必要なライブラリをダウンロードします（初回は1〜3分かかります）

Terminal npm install # node_modules/ フォルダが作成されれば成功 # "npm warn" は無視してOK # "npm ERR!" が出たらエラー → トラブル対処を参照

Terminal npm run dev # 成功すると以下が表示される: ▲ Next.js 16.1.6 (Turbopack) - Local: http://localhost:3000 ✓ Ready in xxxms
💡 停止するには Ctrl + C を押します。

🔧 トラブル対処法

よくある問題と解決方法

⚠ ポート3000が使用中

他のプロセスが3000番ポートを使っている場合に発生します。

対処法:
npm run dev -- -p 3001 で別ポートで起動
または lsof -i :3000 でプロセスを確認して停止
⚠ npm install でエラー

パッケージの依存関係やアクセス権限の問題で発生することがあります。

対処法:
rm -rf node_modules package-lock.json
npm install で再インストール
⚠ ページが真っ白になる

キャッシュやビルドファイルの不整合で発生することがあります。

対処法:
rm -rf node_modules .next
npm install && npm run dev
⚠ ログインできない

localStorageのデータ不整合で発生することがあります。

対処法:
ブラウザ開発者ツール (F12) → Application → Local Storage → Clear All
その後ページを再読込
⚠ Module not found エラー

パッケージが不足している、またはパスが間違っている場合に発生します。

対処法:
npm install を再実行
それでもダメなら node_modules 削除 → 再インストール
⚠ ビルドエラー（型エラー）

TypeScriptの型チェックでエラーが検出された場合に発生します。

対処法:
エラーメッセージのファイル名と行番号を確認 → 該当箇所を修正
as const や型アノテーションの追加で解決するケースが多い
