各要素の算出ロジック

曲データはSpotify APIで検索し、IDだけをDBに保存。マッチング時はIDを比較して一致判定。

📍 1. 地域スコア（最大40pt）

練習やライブは対面で行うため、地域の近さを高く評価。

条件スコア
同じ都道府県 + 同じ市区町村40pt
同じ都道府県のみ25pt
異なる都道府県0pt

🎸 2. 楽器スコア（最大20pt）

補完的な楽器構成（お互いが違う楽器）を高く評価。

条件スコア
完全に異なる楽器（補完的）20pt
一部重複12pt
全て同じ楽器4pt

🎵 3. ジャンルスコア（最大20pt）

好きなジャンルの共通度合い。

スコア = (共通ジャンル数 / 自分のジャンル数) x 20

例: 自分がロック・ポップ・アニソンの3ジャンル、相手がロック・アニソンの場合 → 共通2 / 自分3 x 20 = 13.3pt

🎯 4. スキルスコア（最大15pt）

スキルレベルが近い人同士を高く評価。初心者が上級者とマッチングして萎縮することを防ぐ。

スコア = max(0, 15 - |自分のスキル値 - 相手のスキル値| x 5)
レベル差スコア例
0（同じ）15pt初心者 vs 初心者
110pt初心者 vs 中級者
25pt初心者 vs 上級者
30pt初心者 vs エキスパート

📅 5. スケジュールスコア（最大30pt）

練習可能な曜日・時間帯の一致度。

基本スコア = (共通曜日数 / 自分のスケジュール数) x 30
ボーナス = 共通曜日で時間帯が重複 +5pt/曜日

例: 自分が月・水・土の3日、相手が水・土の場合 → 共通2 / 自分3 x 30 = 20pt（+ 時間帯重複ボーナス）

🎼 6. 曲マッチングスコア（最大35pt）★ コピバン特化

コピバンの核心「同じ曲をやりたい」マッチング。やりたい曲(wantToPlaySongs) + 弾ける曲(canPlaySongs) の一致を評価。

基本スコア = 共通曲数 x 7（最大5曲 = 35pt）
アーティストボーナス = 一致アーティスト数 x 5（曲スコアとの合計で35ptが上限）
一致曲数スコア
5曲以上35pt（上限）
4曲28pt
3曲21pt
2曲14pt
1曲7pt
0曲0pt（アーティストボーナスで加点可能）

曲マッチング計算例

ユーザーA

やりたい曲: 天体観測(BUMP)、花の名(BUMP)、完全感覚Dreamer(ONE OK ROCK)

好きなアーティスト: BUMP OF CHICKEN、ONE OK ROCK

ユーザーB

好きなアーティスト: BUMP OF CHICKEN、Neru

曲一致: 天体観測 + 完全感覚Dreamer = 2曲 → 14pt

アーティスト一致: BUMP OF CHICKEN = 1 → +5pt

合計: 19pt / 35pt
