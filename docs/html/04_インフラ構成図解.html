AWSサービス一覧

Compute ECS Fargate

サーバーレスコンテナ。Next.jsアプリを実行。EC2管理不要。

Database RDS PostgreSQL

マネージドDB。初期はdb.t4g.micro。成長期にAurora移行検討。

Cache ElastiCache

Redisベースのキャッシュ。Spotify APIレスポンス + セッション。

Network ALB

アプリケーションロードバランサー。ECSタスクへの負荷分散。

CDN CloudFront

コンテンツ配信ネットワーク。静的アセットの高速配信 + SSL終端。

Storage S3

静的アセット + 画像アップロード。ライフサイクルポリシーで自動削除。

DNS Route 53

DNS管理。groovematch.jp + サブドメイン。

Auth Cognito

ユーザー認証。ソーシャルログイン対応。5万MAUまで無料。

Security WAF

Webアプリケーションファイアウォール。レートリミット + SQLi/XSS対策。

Monitor CloudWatch

ログ収集、メトリクス監視、アラート。

Email SES

メール送信。トライアル終了通知、パスワードリセット等。

Secret Secrets Manager

APIキー・DB接続情報の安全な保管。自動ローテーション。

Realtime API Gateway WebSocket

リアルタイムチャット用WebSocket接続管理。双方向通信。

Compute Lambda

チャットのWebSocketハンドラー。接続管理・メッセージ送受信をサーバーレスで処理。

セキュリティ多層防御

外側から内側へ、複数レイヤーでアプリケーションを保護。

1

CloudFront + ACM（SSL）

HTTPS強制、DDoS保護（AWS Shield Standard自動適用）

2

WAF（Web Application Firewall）

レートリミット、SQLインジェクション防止、XSS防止、IP制限

3

VPC + Security Groups

パブリック/プライベート/アイソレーテッドのサブネット分離。最小権限のポートアクセス

4

Cognito + JWT

認証・認可。トークンベースのセッション管理。ソーシャルログインの安全な統合

5

RDS Proxy + Secrets Manager

DB接続のプーリング最適化。シークレットの安全な管理と自動ローテーション
