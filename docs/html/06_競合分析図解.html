差別化の3本柱

🎵

曲ベースマッチング

Spotify API連携で「やりたい曲」でマッチング。6要素自動スコアリングで最適な仲間を提案。掲示板の「自力検索」とは一線を画す。

🎯

挫折防止サポート

Duolingo型の練習トラッカー、ストリーク、バッジ、相互フレンド間ランキング。「バンドを組む」というゴールに到達するまでモチベーションを維持。

🎸

コピバン一気通貫

仲間探し → コピバン結成（セットリスト管理・楽器スロット制）→ ライブ出演（イベントボード）まで。バンド結成後もサポートが続く。
