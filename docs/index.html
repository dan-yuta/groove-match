このポータルについて

Groove Match は、コピバン特化のメンバーマッチング&ライブ出演促進アプリです。
ゴールは「コピバンを組んでライブに出ること」。

好きなバンドの曲をコピーしたい初心者が、
「仲間探し → コピバン結成 → ライブ出演」まで一気通貫でたどり着けます。

メインフロー

  • やりたい曲・アーティストで仲間を検索（Spotify API連携）
  • コピバン結成・セットリスト管理
  • ライブイベントへのエントリー

挫折防止サポート

  • Duolingo型練習トラッカー（ストリーク・バッジ）
  • 相互承認型フレンド機能・フレンド間ランキング
  • SNS連携（X / Instagram / LINE / YouTube）
  • ソーシャルログイン（Google / X / Instagram）

Web・iOS・Android のクロスプラットフォーム対応。どのデバイスからでも利用できます。

このポータルでは、仕様・設計・インフラ・ビジネス戦略に関する全ドキュメントを一覧できます。
HTML図解はブラウザでそのまま閲覧、Markdownは GitHub 上で読めます。

まずはデモアプリに触れてみて、そのあと各ドキュメントで詳細を確認してください。
