# Groove Match ドキュメント一覧

`docs/md/` フォルダには、Groove Match（Web + iOS + Android 対応の初心者コピバン向けマッチングアプリ）の仕様・設計に関するドキュメントが格納されています。

**ゴール =「コピバンを組んでライブに出ること」。**
やりたい曲・アーティストで仲間を見つけ、コピバンを結成し、ライブに出るまでを一気通貫でサポートするアプリです。

メインフローは **仲間探し → コピバン結成 → ライブ出演** です。

Duolingo型の練習トラッカー（ストリーク・バッジ・ランキング）、フレンド機能、SNS連携（X/Instagram/LINE/YouTube）は、そのゴールに到達するまでの挫折を防ぐサポート機能として位置づけています。
ソーシャルログイン（Google / X / Instagram）にも対応しており、簡単にアカウント作成が可能です。

初めての方は上から順番に読むことをおすすめします。
