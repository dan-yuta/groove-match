# BandMatch ドキュメント一覧

`docs/` フォルダには、BandMatch（Web + iOS + Android 対応の初心者コピバン向けマッチングアプリ）の仕様・設計に関するドキュメントが格納されています。

### フォルダ構成

```
docs/
├── index.html          … ポータルページ（ブラウザで開く）
├── md/                 … Markdownドキュメント（.md）
└── html/               … HTML図解ファイル（.html）
```

**ゴール =「コピバンを組んでライブに出ること」。**
やりたい曲・アーティストで仲間を見つけ、コピバンを結成し、ライブに出るまでを一気通貫でサポートするアプリです。

メインフローは **仲間探し → コピバン結成 → ライブ出演** です。

Duolingo型の練習トラッカー（ストリーク・バッジ・ランキング）、フレンド機能、SNS連携（X/Instagram/LINE/YouTube）は、そのゴールに到達するまでの挫折を防ぐサポート機能として位置づけています。
ソーシャルログイン（Google / X / Instagram）にも対応しており、簡単にアカウント作成が可能です。

初めての方は上から順番に読むことをおすすめします。

## 読み方ガイド

### ポータルページ（docs/index.html）
- `docs/index.html` をブラウザで開くと、全ドキュメントへのリンク付きポータルが表示されます

### Markdownファイル（docs/md/）
以下のいずれかの方法で閲覧できます。
- **VS Code** で開く → プレビュー機能（Ctrl+Shift+V）で見やすく表示
- **GitHub** にpushすると自動でフォーマットされて表示
- テキストエディタでそのまま開いても読めます

### HTMLファイル（docs/html/）
- ファイルをダブルクリック、またはブラウザにドラッグ&ドロップで開けます
- インターネット接続不要（すべてインライン CSS で完結）
