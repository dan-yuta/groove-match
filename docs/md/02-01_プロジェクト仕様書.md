# BandMatch - 初心者コピバン向けコミュニティアプリ 仕様書

## 概要

**BandMatch** は、好きなバンドの曲をコピーしたい初心者が、同じ曲をやりたい仲間を見つけてコピバンを結成し、コピバンライブに出るまでをサポートするアプリです。

**クロスプラットフォーム（Web / iOS / Android）対応コミュニティアプリ**として提供します。

> **本アプリのゴールは「コピバンを組んでライブに出ること」です。**
> メインフローは **仲間探し → コピバン結成 → ライブ出演** の一気通貫体験であり、練習トラッカーやフレンド機能はそのゴールに到達するまでの挫折を防ぐサポート機能として位置づけています。

**やりたい曲・アーティストベースのコピバンメンバーマッチング**と**ライブ出演の促進**を軸に、月額課金で運営します。

## ターゲットユーザー

### メインターゲット
| 項目 | 内容 |
|------|------|
| 年齢 | 18〜35歳 |
| 性別 | 男女不問 |
| 音楽経験 | 初心者〜中級者（楽器歴0〜3年） |
| 目的 | 好きなバンドの曲をコピーして仲間とライブに出たい |
| 課題 | 同じ曲をやりたい仲間が見つからない / コピバンの組み方が分からない |

### ペルソナ

#### ペルソナ1: 大学生ギタリスト（田中 太郎・20歳）
- 大学入学を機にギターを始めた
- BUMP OF CHICKENの「天体観測」を弾けるようになりたい
- 軽音サークルには入ったが、やりたい曲が合うメンバーが見つからない
- **BandMatchで解決**: 「天体観測」をやりたい人を検索 → 同じ地域のベース・ドラムとマッチング → コピバン結成

#### ペルソナ2: 社会人ドラマー（鈴木 花子・28歳）
- 高校時代に軽音部でドラムを担当、社会人になって再開
- ONE OK ROCKやRADWIMPSのコピーをやりたい
- 社会人バンドサークルを探しているが、日程が合わない
- **BandMatchで解決**: スケジュールマッチングで週末練習可能なメンバーを発見 → ワンオクコピバン結成

#### ペルソナ3: 初心者ベーシスト（山田 健太・24歳）
- ベースを始めて3ヶ月、まだ自信がない
- アニソン（けいおん!、ぼっち・ざ・ろっく!）のコピーがしたい
- 経験者ばかりの場所だと萎縮してしまう
- **BandMatchで解決**: 同じ初心者レベル＋アニソン好きでマッチング → 初心者同士のコピバンで安心してライブを目指す

## 主要機能

> 機能の優先順位: **1. マッチング（仲間探し）→ 2. コピバン結成（セットリスト管理）→ 3. ライブ出演（イベントボード）** がメインフロー。
> 練習トラッカー・フレンド・ランキングは「挫折防止のサポート機能」です。

### 1. ユーザー登録・認証
- メールアドレス + パスワードによるモック認証
- マルチステップ登録フォーム（基本情報 → 音楽プロフィール → 地域・スケジュール）
- パスワード要件: 8文字以上、英字+数字必須
- localStorage ベースのセッション管理
- クロスタブ同期対応

### 2. プロフィール管理
- 基本情報: 名前、ニックネーム、アバター、自己紹介
- 音楽情報: 楽器（複数可）、スキルレベル、ジャンル（複数可）
- コピバン情報: コピーしたいアーティスト、やりたい曲（wantToPlaySongs）
- 活動情報: 都道府県、市区町村、練習スケジュール、影響を受けたアーティスト
- 表示/編集モード切り替え

### 3. コピバンマッチング
**マッチングアルゴリズム（最大160点→100点に正規化）:**

| 項目 | 最大スコア | ロジック |
|------|-----------|---------|
| 地域 | 40pt | 同県25pt、同市40pt |
| 楽器 | 20pt | 補完的=20pt、一部重複=12pt、全重複=4pt |
| ジャンル | 20pt | 共通ジャンル比率 × 20 |
| スキル | 15pt | レベル差が小さいほど高スコア |
| スケジュール | 30pt | 共通曜日比率 × 30 + 時間帯重複ボーナス |
| 曲マッチング | 35pt | やりたい曲の一致数 × 7（最大5曲分）+ 好きなアーティスト一致ボーナス |

**曲マッチング詳細:**
- やりたい曲（wantToPlaySongs）が1曲一致するごとに7pt（最大35pt）
- 好きなアーティストが一致する場合、アーティストごとに+3ptのボーナス（曲マッチング35ptの内数）
- 同じアーティストの別の曲をやりたい場合も親和性が高いため+2pt

**フィルター:** やりたい曲、アーティスト、ジャンル、楽器、地域、スキルレベル

### 4. コピバン管理
- コピバン作成（名前、説明、コピー対象アーティスト/曲、ジャンル、地域）
- セットリスト管理（コピーする曲の一覧・順序管理）
- 楽器スロット制（空きポジション表示）
- メンバー管理（リーダー/メンバー権限）
- 募集中フラグ
- コピバン詳細・編集画面

### 5. ライブイベントボード
- イベント一覧（フィルター: 地域、日付、初心者歓迎、ジャンルタグ）
- イベント詳細（会場、日時、料金、登録バンド一覧）
- バンド登録機能
- 初心者歓迎タグ

### 6. コミュニティ
- タイムライン（投稿一覧）
- 投稿タイプ: 一般、練習ログ、マイルストーン、質問
- いいね・コメント機能
- 練習ログ（練習時間記録）
- マイルストーン（初セッション、初ライブ、バンド結成等）

### 6a. 挫折防止サポート機能（Duolingo型ゲーミフィケーション）
> メインゴール「コピバンを組んでライブに出る」に到達するまでの挫折を防ぐサポート機能群です。

- **練習トラッカー** - 曲ごとの練習時間記録、ストリーク（連続練習日数）、週間目標
- **フレンド＆ランキング** - フレンド間の練習状況共有、週間ランキング、応援機能
- **バッジ＆マイルストーン** - 12種類のバッジ（ストリーク、累計時間、初ライブ等）
- **通知システム** - 練習リマインド、ストリーク危機、フレンド活動、ランキング変動
- **SNS連携** - X / Instagram / LINE / YouTube アカウント連携

## セキュリティ対策

| 対策 | 実装箇所 | 説明 |
|------|----------|------|
| 入力バリデーション | `validators.ts` | メール、パスワード、名前等の形式チェック |
| XSS防止 | `security.ts` + React JSX | HTMLエスケープ + Reactの自動エスケープ |
| CSRFトークン | `security.ts` | sessionStorageベースのトークン管理 |
| セキュリティヘッダー | `next.config.ts` | X-Frame-Options, CSP, X-Content-Type-Options |
| 認証ガード | `ProtectedRoute` | 未認証時はログインページへリダイレクト |
| 管理者権限チェック | `ProtectedRoute` (adminOnly) | 非管理者はダッシュボードへリダイレクト |
| localStorage名前空間 | `storage.ts` | `bandmatch-` プレフィックス |
