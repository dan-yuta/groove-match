# Groove Match - 競合分析

> **本アプリのゴールは「コピバンを組んでライブに出ること」です。**
> メインフローは「仲間探し → コピバン結成 → ライブ出演」。
> 練習トラッカーやフレンド機能は、そのゴールに到達するまでの挫折を防ぐサポート機能として位置づけています。

---

## 1. 競合サービス概要

### OURSOUNDS（https://oursounds.net/）

| 項目 | 内容 |
|------|------|
| 形態 | バンドメンバー募集 掲示板（Web） |
| 運営歴 | 2005年〜（20年以上） |
| 料金 | 無料 |
| 主な機能 | 募集記事の投稿・検索、メンバー検索、メッセージ、お気に入り |
| 検索方法 | 都道府県別、テーマ別（初心者・シニア等）で絞り込み |
| ターゲット | 全年代・全レベル |
| UI | 従来型の掲示板スタイル |

**強み:**
- 無料で利用可能
- 20年以上の運営実績でユーザー数が多い
- 不正ユーザーの徹底排除に注力

**弱み:**
- UIが古い掲示板スタイルで、若年層には馴染みにくい
- マッチング機能なし（自力で検索するのみ）
- 挫折防止やモチベーション維持の仕組みがない
- バンド結成後のサポート（セットリスト管理等）がない
- 用が済んだら使わなくなる（継続利用の動機がない）

---

### ROCKJOINT（https://rockjoint.link/）

| 項目 | 内容 |
|------|------|
| 形態 | オフラインイベント運営 + マッチングアプリ |
| 実績 | 累計4,601名参加 |
| 料金 | イベント参加費（詳細非公開） |
| 主な機能 | バンドメンバー募集交流会、バンドサークル「コピフェス」、ライブイベント |
| 展開地域 | 東京・大阪・京都・神戸・横浜・福岡 |
| マッチングアプリ | 1,000名以上登録 |

**強み:**
- リアルイベントでの対面交流（実際に会える）
- ライブ出演機会を直接提供
- プロアーティストとのコラボ実績

**弱み:**
- 対面前提のため地方ユーザーには弱い
- オンライン機能が弱い（アプリは補助的）
- 継続的な関係構築の仕組みが薄い
- イベント日程に合わせる必要がある

## 4. Groove Match の差別化ポイント

### 4.1 差別化の3つの柱

#### 柱1: 曲ベースのスマートマッチング（仲間探し）

- **「やりたい曲が同じ仲間が、すぐ見つかる」** がコアバリュー
- Spotify API連携で曲を検索・登録 → 同じ曲をやりたい人がすぐ見つかる
- 6要素スコアリング（曲マッチングが最重要ウェイト）で「探す手間」をゼロに
- マッチングアルゴリズムの詳細は [02-01_マッチングアルゴリズム.md](./02-01_マッチングアルゴリズム.md) を参照

#### 柱2: コピバン結成→ライブまで一気通貫

- 曲マッチングで同じ曲をやりたい仲間を見つける
- コピバン作成 + セットリスト管理でチームを組む
- イベントボードでコピバンライブを見つけてエントリー
- 1つのアプリで完結（複数サービスを行き来しなくてよい）

#### 柱3: Duolingo型挫折防止エコシステム（サポート機能）

- **練習トラッカー**: 曲ごとの練習時間を記録。ストリークで継続を可視化
- **バッジ＆マイルストーン**: 12種類のバッジで成長を実感
- **フレンドランキング**: 相互フレンド間の週間練習ランキングで切磋琢磨
- **通知システム**: 練習リマインド、ストリーク危機通知でモチベーション維持
- **ソーシャルログイン + SNS連携**: 登録・発信のハードルを最小化

### 4.2 競合にないもの（Groove Matchだけの強み）

| 差別化要素 | 競合との違い |
|-----------|-------------|
| **コピバン特化** | 競合はオリジナル・コピー混在。コピバン向けの専用機能がない |
| **Spotify API連携の曲ベースマッチング** | 競合には曲ベースの検索/マッチング機能がない |
| **セットリスト管理** | 競合にはバンド管理機能自体がない |
| **Duolingo型ゲーミフィケーション** | 競合2社にはこの概念がない |
| **相互承認型フレンド + ランキング** | OURSOUNDSにはフレンド機能なし、ROCKJOINTはイベント限定 |
| **招待コード**（トライアル延長） | 競合に招待・紹介の仕組みがない |
| **クロスプラットフォーム** | OURSOUNDSはWeb掲示板のみ |
| **初心者特化** | 競合は全レベル混在で初心者が萎縮しがち |

## 5. 今後強化すべき機能（競合との差をさらに広げる）

| 機能 | 優先度 | 差別化効果 |
|------|--------|-----------|
| プッシュ通知 | 高 | マッチング通知・チャット通知をリアルタイム配信 |
| 練習マッチング | 高 | 「今週末セッションしたい人」のリアルタイム募集 |
| 成長トラッキング | 中 | 練習時間の推移グラフ、スキルレベルの自動更新提案 |
| レコメンドエンジン | 中 | 行動履歴ベースのおすすめ |
| 動画プロフィール | 低 | 演奏力が事前に分かる自己紹介・演奏動画 |
| オンラインセッション | 低 | WebRTCでリモートセッション |

## 6. キャッチコピー案

### メインコピー
> **「やりたい曲で仲間を見つけて、コピバン組んで、ライブに出よう」**

### サブコピー候補
- 「『天体観測』やりたい人、3人見つかりました -- Groove Matchで、コピバンライブデビューしよう」
- 「好きな曲でつながる → コピバン結成 → ライブ出演。全部このアプリで」
- 「掲示板で探す時代は終わり。やりたい曲を登録するだけで仲間が見つかる」

### 競合との違いを一言で
- **vs OURSOUNDS:** 「曲名で検索できる。同じ曲をやりたい人だけが見つかる」
- **vs ROCKJOINT:** 「イベントを待たなくていい。やりたい曲を登録するだけで仲間が見つかる」
